- описание бота:
    - отправляешь ему
        - либо только интересуемый кусок: `most significant bit`
        - либо кусок текста, в котором выделен интересуемый фрагмент: `most *significant* bit`
    - он отправляет в ответ
        - подборку картинок

          тут: https://youtu.be/77eIEv0kDsQ?t=3m51s)

      конкретного места с учетом контекста
